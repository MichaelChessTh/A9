## 🚀 Features

- **Blazing Fast Messaging**: Real-time message delivery with near-zero latency using robust architectures.
- **Secure & Private**: Advanced encryption ensuring your conversations stay between you and your recipients.
- **Cross-Platform**: Stay connected anywhere—Android, iOS, and macOS seamlessly integrated.
- **Rich Media**: Share photos, videos, voice messages, and files instantly, anywhere, anytime.
- **Voice & Video Calls**: High-quality real-time communications powered by leading technologies.
- **Modern UI/UX**: Sleek, intuitive, and dynamic interfaces that feel native to your device.

## 🛠 Tech Stack

- **Frontend**: Flutter (Dart)
- **Backend**: Firebase (Auth, Firestore, Realtime Database, Storage)
- **Communications**: Zego Cloud (Video/Audio Calls)

## 📥 Installation

Navigate to the `dist/` directory in our releases or build from source:

```bash
# Clone the repository
git clone https://github.com/MichaelChessTh/A9.git

# Enter the project directory
cd GoogleChat

# Install dependencies
flutter pub get

# Run the app
flutter run
```
